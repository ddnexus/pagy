# Console

Allows you to interact with Pagy in an irb/pry environment without an app, providing a pre-configured stubbed environment for you.

You can use a few methods to create a simple collection paginated with the :offset paginator or to set the parameters as needed.

Console
require 'pagy/console'
=> true
