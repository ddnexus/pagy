#

## :icon-list-unordered: urls_hash

---

`urls_hash` returns the `:first`, `:previous`, `:next`, `:last` non-`nil` URLs hash.

!!!success It works with all paginators
!!!

```ruby Controller
urls_hash = @pagy.urls_hash(**options)
```

=> Object
